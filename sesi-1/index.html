Praktek Pemrograman Web

Sesi 1

Praktek Pemrograman web 1 akan membahas tentang cara menampilkan halaman web dengan menggunakan HTML, CSS, dan JavaScript. mahasiswa diharapkan mampu memahami sintaks dasar dalam menampilkan halaman web, dan dengan pemahaman tersebut menuliskan hasil kreasinya sendiri

  • HTML
  • CSS
  • JavaScript

pada sesi 1 mahasiswa mengenali dulu struktur dasar HTML beserta sintaks dasarnya.

HTML

  1. Pengenalan HTML
  2. Tabel
  3. Form

CSS

cascading style sheet (CSS) merupakan standar untuk mengatur tampilan yang ada pada suatu berkas HTML. CSS bukan bahasa pemograman tetapi merupakan bahasa deklaratif yang dinyatakan langsung untuk menyatakan suatu tampilan, tidak terdapat logika, percabangan, ataupun perulangan.

Javascript

Javascript merupakan bahasa pemograman yang berjalan disis browser, diisi pengguna, atau di peramban bukan server
